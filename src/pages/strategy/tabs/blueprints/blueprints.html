What is this page?
This page shows player's ships with required remodeling materials, such as blueprints.
What do gray materials mean?
They indicate you have already used those materials to get corsponding remodeled ships.
What the differences about All Ships view?
All Ships view shows you all the ships implemented by KanColle game for now require blueprints materials on remodeling.
It also tries to show you already used materials and total needs by checking ships you owned. Different from Owned Ships view, some columns changed:
  • ID termed Master ID, identifies this type of ship at Ship Library;
  • Level is the required level to next remodel of this ship;
  • Requirements column also shows you the ship name of next remodel.
Why my data seem weird or wrong after game maintenance?
Sometimes KanColle devs may not update remodel info in time or put dummy data into API result if multi maintenances scheduled shortly. To fix such issues after in-game data stable, you might try to refresh our remodel info cache by clicking 'Rebuild ship remodel information caches' link at Profile page.
